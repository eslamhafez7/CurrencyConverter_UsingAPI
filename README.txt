Currency Converter Project

Description:
----------------
The Currency Converter Project is a simple web application that allows users to convert an input
amount from one currency to several other currencies. The user can enter an amount, select a base
currency, and then click the "Calculate" button to see the equivalent values in various foreign
currencies.

Features:
------------
- Convert an input amount to multiple currencies.
- Supports various currencies, including EGP, KWD, SAR, AED, EUR, JPY, GBP, AUD, BRL, and RUB.
- Real-time currency conversion using live exchange rates from the CurrencyFreaks API.

CurrencyFreaks API Integration:
-------------------------------------
The Currency Converter Project uses the CurrencyFreaks API to fetch live exchange rates for currency
conversion. The API provides up-to-date exchange rate data for various currencies, enabling real-time
conversions.

API Endpoint: https://api.currencyfreaks.com
Documentation: [Link to CurrencyFreaks API Documentation]

Usage:
---------
1. Open the "index.html" file in your web browser.
2. Enter the amount you wish to convert in the input field labeled "Enter Number."
3. Select the base currency from the dropdown list.
4. Click the "Calculate" button to see the converted amounts below.

Files and Directories:
----------------------
- index.html: The main HTML file that contains the currency converter interface.
- styles.css: The CSS file to style the currency converter interface.
- script.js: The JavaScript file to handle currency conversion and user interactions.
